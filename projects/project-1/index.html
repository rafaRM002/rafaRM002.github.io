BandCoord

BandCoord es una aplicación web para la gestión completa de bandas de música, con foco en agrupaciones de Semana Santa. Permite organizar ensayos, actuaciones y comunicación con los miembros, gestionar disponibilidad, instrumentos, secciones, y registrar fechas clave mediante un calendario interactivo. Además, incluye un sistema de mensajería y almacenamiento de partituras y archivos multimedia. Desarrollado con Laravel, React y TailwindCSS.
