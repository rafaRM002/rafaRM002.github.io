Inventory Manager – Team Project at Qualica RD

Durante mis prácticas profesionales en Qualica RD, participé en un proyecto colaborativo con SantanaDV y ManuelGonzalez709 para desarrollar una aplicación multiplataforma de gestión de inventario y tareas en almacenes. Backend con Spring Boot y frontend con React.

During my internship at Qualica RD, I participated in a team project with SantanaDV and ManuelGonzalez709 to develop a cross-platform inventory and task management application. Backend in Spring Boot, frontend in React.

Objetivos / Goals

  • Diseñar una aplicación ágil y accesible desde dispositivos móviles y escritorio.
  • Facilitar colaboración mediante control del stock y gestión de tareas en tiempo real.
  • Aplicar metodologías ágiles y buenas prácticas de equipo usando Git.
  • Design an agile application accessible from both mobile and desktop devices.
  • Facilitate collaboration through real-time stock control and task management.
  • Apply agile methodologies and team best practices using Git.

Tecnologías / Technologies

  • Backend: Spring Boot (Java).
  • Frontend: React (JavaScript).
  • Control de versiones: Git y GitHub.
  • Otros: CSS para diseño y adaptación.
  • Backend: Spring Boot (Java).
  • Frontend: React (JavaScript).
  • Version control: Git and GitHub.
  • Other: CSS for responsive design.

Funcionalidades / Features

  • Gestión de inventario y tareas:
    • Alta, edición y baja de productos.
    • Control de stock en tiempo real.
    • Asignación y seguimiento de tareas.
  • Sistema de usuarios y roles:
    • Autenticación segura y panel de administración.
  • Inventory and task management:
    • Create, update, and delete products.
    • Real-time stock monitoring.
    • Task assignment and tracking.
  • User and role system:
    • Secure authentication and admin panel.

Resultados / Outcomes

Este proyecto reforzó mis habilidades Full Stack y trabajo en equipo profesional. Aporté soluciones backend y frontend gracias a mi formación previa en DAM.

This project strengthened my Full Stack skills and teamwork in a professional environment. I contributed both backend and frontend solutions thanks to my DAM training.
