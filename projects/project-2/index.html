Gestor de Inventario – Proyecto en Equipo en Qualica RD

Durante mis prácticas profesionales en Qualica RD, participé en un proyecto colaborativo con SantanaDV y ManuelGonzalez709 para desarrollar una aplicación multiplataforma de gestión de inventario y tareas en almacenes. Backend con Spring Boot y frontend con React.

Objetivos principales

  • Diseñar una aplicación ágil y accesible desde dispositivos móviles y escritorio para gestionar inventario y tareas.
  • Facilitar la colaboración entre el personal mediante control en tiempo real del stock, gestión de tareas y notificaciones.
  • Aplicar metodologías ágiles y buenas prácticas colaborativas, usando control de versiones (Git) y trabajo en equipo profesional.

Tecnologías empleadas

  • Backend: Spring Boot (Java) para una API REST segura y eficiente.
  • Frontend: React (JavaScript) para una interfaz intuitiva y adaptable.
  • Control de versiones: Git y GitHub para coordinar el trabajo en equipo.
  • Otros: CSS para diseño y enfoque multiplataforma.

Funcionalidades destacadas

  • Gestión de inventario y tareas:
    • Creación, modificación y eliminación de productos.
    • Control en tiempo real del stock disponible.
    • Asignación y seguimiento de tareas para el personal del almacén.
  • Sistema de usuarios y roles:
    • Autenticación segura con roles diferenciados para trabajadores y administradores.
    • Panel de administración para supervisar movimientos de inventario y tareas.
  • Colaboración y trabajo en equipo:
    • Coordinación con SantanaDV y ManuelGonzalez709, asignando responsabilidades según fortalezas.
    • Uso de metodologías ágiles y reuniones periódicas para seguimiento y mejora continua.
  • Pruebas y despliegue:
    • Implementación y pruebas en entorno real de empresa para garantizar estabilidad y funcionalidad.

Resultados y aprendizajes

Este proyecto fue un gran avance en mi formación profesional, consolidando mis habilidades como desarrollador Full Stack y mejorando mi capacidad para trabajar en equipo en un entorno empresarial real.

Gracias a mi formación previa en DAM, aporté soluciones integrales en backend y frontend, asegurando calidad y buenas prácticas.
