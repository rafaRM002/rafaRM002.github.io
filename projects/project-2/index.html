Gestor de Inventario – Proyecto en Equipo en Qualica RD

Durante mis prácticas profesionales en Qualica RD, participé en un proyecto colaborativo con SantanaDV y ManuelGonzalez709 para desarrollar una aplicación multiplataforma de gestión de inventario y tareas en almacenes. Backend con Spring Boot y frontend con React.

Objetivos

  • Diseñar una aplicación ágil y accesible desde dispositivos móviles y escritorio.
  • Facilitar colaboración mediante control del stock y gestión de tareas en tiempo real.
  • Aplicar metodologías ágiles y buenas prácticas de equipo usando Git.

Tecnologías

  • Backend: Spring Boot (Java)
  • Frontend: React (JavaScript)
  • Control de versiones: Git y GitHub
  • Otros: CSS para diseño y adaptación

Funcionalidades

  • Gestión de inventario y tareas:
    • Alta, edición y baja de productos.
    • Control de stock en tiempo real.
    • Asignación y seguimiento de tareas.
  • Sistema de usuarios y roles:
    • Autenticación segura y panel de administración.

Resultados

Este proyecto reforzó mis habilidades Full Stack y trabajo en equipo profesional. Aporté soluciones backend y frontend gracias a mi formación previa en DAM.

Nota: Proyecto desarrollado en prácticas DAW con SantanaDV y ManuelGonzalez709 usando Spring Boot y React.
