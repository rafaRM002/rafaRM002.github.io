Gestor de Inventario – Proyecto en Equipo en Qualica RD

Descripción del proyecto:

Durante mis prácticas profesionales en Qualica RD, participé en un proyecto colaborativo junto a los compañeros SantanaDV y ManuelGonzalez709. El objetivo fue desarrollar una aplicación multiplataforma orientada a la gestión integral de inventario y tareas dentro del almacén. La solución combina un backend desarrollado en Spring Boot con un frontend dinámico y responsive construido en React. Este proyecto formó parte del ciclo formativo de Desarrollo de Aplicaciones Web (DAW).

Objetivos principales

  • Diseñar una aplicación ágil y accesible desde dispositivos móviles y de escritorio para gestionar el inventario y las tareas asociadas.
  • Facilitar la colaboración entre el personal mediante el control en tiempo real del stock, gestión de tareas y notificaciones.
  • Aplicar metodologías ágiles y buenas prácticas de desarrollo colaborativo, haciendo uso de sistemas de control de versiones (Git) y trabajo en equipo profesional.

Tecnologías empleadas

  • Backend: Spring Boot (Java) para una API REST segura y eficiente.
  • Frontend: React (JavaScript) para crear una interfaz intuitiva y adaptable a diferentes dispositivos.
  • Control de versiones: Git y GitHub para coordinar el trabajo en equipo.
  • Otros: CSS para el diseño y enfoque multiplataforma (móvil y escritorio).

Funcionalidades destacadas

  • Gestión de inventario y tareas:
    • Creación, modificación y eliminación de productos.
    • Control en tiempo real del stock disponible.
    • Asignación y seguimiento de tareas para el personal del almacén.
  • Sistema de usuarios y roles:
    • Autenticación segura con roles diferenciados para trabajadores y administradores.
    • Panel de administración para supervisar y consultar movimientos de inventario y tareas.
  • Colaboración y trabajo en equipo:
    • Coordinación con SantanaDV y ManuelGonzalez709, asignando responsabilidades según las fortalezas de cada miembro.
    • Uso de metodologías ágiles y reuniones periódicas para seguimiento y mejora continua.
  • Pruebas y despliegue:
    • Implementación y pruebas en un entorno real de empresa, garantizando la estabilidad y funcionalidad de la aplicación.

Resultados y aprendizajes

Este proyecto representó un gran avance en mi formación profesional. Consolidé mis habilidades como desarrollador Full Stack, mejoré mi capacidad para trabajar en equipo y enfrenté retos reales propios de un entorno empresarial. Gracias a mi formación previa en el Grado Superior de Aplicaciones Multiplataforma (DAM), pude aportar una visión integral y soluciones que abarcaban tanto el backend como el frontend, asegurando calidad y buenas prácticas en el producto final.

Nota: Proyecto desarrollado en colaboración con SantanaDV y ManuelGonzalez709 durante las prácticas del ciclo DAW en Qualica RD, utilizando Spring Boot y React.
